# My .dotfiles !

I have dotfiles for both windows and linux. View the appropriate directory for details.

## Bootstrap

### Windows

* Install chocolatey
* `choco install git make gsudo`
* Clone in `~\Repos\dotfiles`
* `make windows`

### Linux

* Install git
* Clone in `~/dotfiles`
* `make linux` (in git bash)
