# My .dotfiles !

They are designed to be used with the latest Ubuntu version running in WSL 2.

## Installation
* `git clone git@github.com:maxencefrenette/dotfiles.git`
* `./dotfiles/install.sh`
* Configure ssh keys manually

## Inspiration
